go语言用途

Go 语言被设计成一门应用于搭载 Web 服务器，存储集群或类似用途的巨型中央服务器的系统编程语言。
对于高性能分布式系统领域而言，Go 语言无疑比大多数其它语言有着更高的开发效率。它提供了海量并行的支持，
这对于游戏服务端的开发而言是再好不过了。

go语言环境安装

go安装包安装

安装包下载地址为：https://golang.org/dl/。
或者到这个中文网站下载https://studygolang.com/dl；
安装msi文件到windows电脑上面，默认情况下.msi文件会安装在 c:\Go 目录下。
你可以将 c:\Go\bin 目录添加到 PATH 环境变量中。添加后你需要重启命令窗口才能生效。

go语言的数据结构

变量来源于数学，是计算机语言中能储存计算结果或能表示值抽象概念。变量可以通过变量名访问。
Go 语言变量名由字母、数字、下划线组成，其中首个字母不能为数字。
声明变量的一般形式是使用 var 关键字。

方法一，指定变量类型

var v_name v_type
v_name = value

方法二，根据值自行判定变量类型

var v_name = value

方法三，省略var

注意：=左侧的变量不应该是已经声明过的；否则会导致编译错误。
